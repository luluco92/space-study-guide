Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: switches to this branch in the console
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes to repo
  • git commit -m "comment": commit changes
  • git pull origin main: merge 'main' version into 'origin' version, This is to pull down the most recent main version to local computer
  • git push origin feature/add-html: merge 'origin' version into 'feature/add-html' version, adds and overrites files. Pushes our changes from computer to github repo.
