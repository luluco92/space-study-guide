Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Inline CSS: Using style="color:red;" attribute in an element. Good for one-offs.
  • Internal CSS: having a <style> element in the <head> of a html page. Limited to just one HTML page, but applies to all elements of that type/.class/id. e.g. <style> h1 {color: red;} </style>
  • External CSS: Using a separate css text file that can be used by multiple HTML pages. It works the same as internal CSS. insert <link rel="stylesheet" href="./assets/style.css" /> into the <head>.
  • NOTE: CSS will override basic parameters like width/height.

Git

  • git status: checks what branch we are currently on
  • git checkout branch-name: switches to this branch in the console
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes to repo
  • git commit -m "comment": commit changes
  • git pull origin main: merge 'main' version into 'origin' version, This is to pull down the most recent main version to local computer
  • git push origin feature/add-html: merge 'origin' version into 'feature/add-html' version, adds and overrites files. Pushes our changes from computer to github repo.
